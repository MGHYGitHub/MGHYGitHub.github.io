PicX еӣҫеәҠ







PicX жҳҜдёҖж¬ҫеҹәдәҺ GitHub API ејҖеҸ‘зҡ„е…·жңү CDN еҠ йҖҹеҠҹиғҪзҡ„еӣҫеәҠз®ЎзҗҶе·Ҙе…·

еңЁзәҝдҪҝз”Ёе…ҘеҸЈ https://picx.xpoet.cn

зҒөйӯӮжӢ·й—®пјҢдҪ дёәеӣҫеәҠй—®йўҳзғҰжҒјиҝҮеҗ—?

  • з”Ё HexoгҖҒVuePressгҖҒHugo зӯүйқҷжҖҒеҚҡе®ўеҶҷж–Үз« пјҢдёҚзҹҘеӣҫзүҮдҝқеӯҳеҲ°е“ӘйҮҢеҺ»вҖҰ
  • зү№ж„ҸиҠұй’ұз§ҹдә‘жңҚеҠЎеҷЁжүҳз®ЎеӣҫзүҮпјҢеӨӘиҙөеҲ’дёҚжқҘпјҢиҖҢдё”дёҠдј й…ҚзҪ®еҘҪз№ҒзҗҗвҖҰ
  • зҪ‘дёҠеӨҚеҲ¶зҡ„еҝғд»ӘеӣҫзүҮзҡ„й“ҫжҺҘпјҢз”ЁзқҖз”ЁзқҖжҹҗдёҖеӨ©е°ұеӨұж•ҲдәҶвҖҰ
  • дҪҝз”Ёе…¶д»–зҡ„д»ҳиҙ№еӣҫеәҠпјҢйҖҹеәҰж…ўпјҢе®№йҮҸе°ҸпјҢиҝҳйҷҗж—¶гҖҒйҷҗжөҒйҮҸвҖҰ
  • жғіжүҫдёҖж¬ҫзңҹжӯЈе…Қиҙ№гҖҒзЁіе®ҡгҖҒдёҚйҷҗе®№йҮҸгҖҒи®ҝй—®йҖҹеәҰиҝҳеҫҲеҝ«зҡ„еӣҫеәҠвҖҰ
  • вҖҰвҖҰ

йӮЈд№ҲпјҢеҝ«жқҘиҜ•иҜ• PicX еӣҫеәҠ еҗ§пјҢдё“дёәжҠҖжңҜеҚҡдё»жү“йҖ пјҢи°Ғз”Ёи°ҒзҹҘйҒ“~

еҸӘйңҖйҖүжӢ©дёҖдёӘ GitHub д»“еә“дҪңдёәеӣҫеәҠпјҢз„¶еҗҺеңЁ PicX е®ҳзҪ‘ е®ҢжҲҗ Token з»‘е®ҡе’Ңзӣёеә”й…ҚзҪ®е°ұиғҪдҪҝз”ЁдәҶгҖӮ

PicX еӣҫеәҠ жөҸи§ҲеҷЁеңЁзәҝдҪҝз”ЁпјҢе…ҚдёӢиҪҪ&е®үиЈ…пјҢеҰӮжӯӨз®ҖеҚ•гҖӮ
