什么是函数?

在大家的认知里面,函数可能是数学领域中的一个名词,它可能是这样的

但是数学领域中的函数与编程语言中的函数是完全不同的

数学函数 ≠ 编程函数

首先,int表明了函数的返回值类型为整数类型

这是由语言标准中规定的,我们不可以写成别的词

int是integer整数的缩写

main是函数的名称,我们称之为主函数main后面的括号()就是用来输入参数的[目前这个参数为空]

这时我们就可以对应到了第一个特征

函数是可以输入一些值的,当然也可以不输入

函数是需要给{}花括号包括的,{}花括号内部就是函数题,我们可以在{}的内部做一些处理或者操作

所以花括号{}是绝对不能被省略的

而这个正好对应第二个特征

函数内部可以处理一些事情

而上面那段代码所做的就是将 Hello World显示在屏幕上

return之后跟着的是函数的返回值,这里返回了一个 0 |0是一个整数,和前面的int是对应的

这个就对应到了第三个特征

函数可以返回一些值

主函数

所有C语言代码都有起始入口,这个入口就是主函数main.

进入主函数后,才能由主函数调用其他函数.

每个C语言代码,有且只能有一个主函数.

我们在把代码修改一下

当程序运行的时候

  1. 首先会进入主函数main
  2. 接着在main中调用刚刚定义好的add函数

我们传入了两个值,分别为2和3给add函数,通过这样的修改程序就可以正确被运行了

虽然add函数的确被main函数成功调用了,但是编译器是怎么理解add这个名称的呢?

编译器会从代码开始按照从上往下的顺序阅读代码

  1. 编译器首先看到了一个函数的定义,描述了一个叫做add的函数
  2. 接着在main中需要使用到add,由于编译器已经知道了add的定义,因此编译器可以正常编译通过

但是如果将函数定义和函数调用反过来呢?

首先编译器看到了add的名称,编译器会很疑惑add是什么?

编译器无法理解add究竟是什么,因此编译器将报错并停止编译

编译器将报错并停止编译

add函数成功被主函数调用,那么自然的我们就会想到:

  • 主函数又是谁调用的呢?
  • 主函数的返回值又必须是int吗?

主函数在程序开始的时候是被自动调用的,不需要在程序内主动调用主函数

而主函数的返回值会返回给调用这个程序的程序

C语言标准中规定主函数有返回值,且必须是int,如果程序正常的结束,一遍我们将返回值设置为 0

什么是变量

回到之前的代码

在add函数计算完毕之后,需要有一个东西来接收add函数返回回来的值

所以我们在add前面声明了一个int整形变量result

result只是我们起的一个名字而已,当然你也可以起别的名字

我们把add函数返回回来的 5 装进了result

因此变量result里面装的值就是 5

= 在C语言中是赋值运算符的意思

它的功能是将右边的值装进左边的变量, 和函数一样,这里的 = 和数学里面的 = 也是有很大的不同的

它并不是相等的意思

赋值运算符是将右边的值装进左边的变量的一种运算符

再来分析一次这个流程

  1. 函数开始运行
  2. 在主函数中调用add函数,传入2和3的函数
  3. add函数将2和3进行相加并返回了相加的结果
  4. 之后通过赋值运算将结果5赋值给了变量result

问题:可不可以将 int result这一句去掉呢?

答案是:

NO

因为这一行代码是声明变量的语句

变量必须先声明之后才能被使用

标识符 关键词

在前面的代码中有我们自己命名用于指代某一个实体的名称

eg: add ``result函数的参数ab都是一个标识符

标识符是由我们自己命名的一个特殊标识

用来表示一个变量函数或者其他实体的名称,并且要让编译器能够识别标识符,必须进行声明或者定义

eg:

  • add被定义成了一个函数
  • resylt ab被声明成了一个整形类型的变量

如果编译器遇到一个未经定义或者声明的标识符,将会无法理解这个标识符具体是一个什么实体的名称,因此会报错

问题:int是标识符吗?

答案是:

NO

首先int并不是我们随意命名的

其次int并不是任何一个实体的名称

int是C语言中的一个关键词,关键词是在语言标准中规定的,并且在代码中有着特殊的意义和用途,因此关键词是不能作为一个标识符来进行使用的

这个表格列举了C语言中所有的关键词

它们全都不能作为标识符进行使用

什么是常量

有变量就会有不变的量,像二三这种数值就不属于不变的量,我们称为常量

常量和变量是不一样的,它并不需要被声明

因为一旦常量写出来,我们就可以立刻知道像二三这种数值它是属于整数int类型的常量

同样的字符串字面常量也不需要被声明

eg: “Hello World”

被””包括的,我们就认为它是一个字符串,以区别于数值变量

变量我们可以通过赋值来更改,常量是不能更改的,所以你不能对它进行赋值

像这样直接对常量的赋值的写法是完全错误的

printf 函数

printf也是一个函数,但是这个并不是我们自己定义的函数

而是系统自带的函数

我们把字符串hello world传给了printf函数,运行代码后,我们在屏幕上看到了这行字符串

printf函数的作用是将字符串输出到控制台上

printf由单词print打印和单词fmt格式的首字母f组成,意为格式化打印

printf除了直接传入一个字符串,它还可以通过占位符的方式进行打印,康康如下代码:

%d就是一个占位符,它是整形占位符,最终printf打印的结果会使用result的值代替百分号d这个位置

举个🌰

我们想用printf函数打印一个整数,可以这样写

printf("%d",整数int);

我们想用printf函数打印两个整数,可以这样写

printf("%d %d",整形int1,整形int2);

通过这两个🌰,我们就可以总结出printf的公式

printf的第一个参数必须是字符串,字符串可以包含多个占位符,而占位符是使用%在装换操作来进行表示的

eg: 整形int的占位符为%d,后面的替换参数会依次替换前面的占位符

通过这个代码我们再次解析一下printf的规律

include 命令

还是这串代码,我们发现在代码的最开始有一个#include语句

这个语句是用来引入头文件的,例如printf是系统自带的函数,前面我们也提到了,要调用函数就必须先定义好函数,这样编译器才能够理解printf到底是什么

而printf的定义就写在这个文件之中,用#include命令可以将文件中的代码复制到我们的代码中,这样我们才能够使用printf函数,当然这个文件里面并没有定义print函数,但是它里面有printf函数的函数声明,在当前的阶段,你完全可以理解这个文件里面写了printf函数的函数定义
